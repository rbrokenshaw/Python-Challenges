<h1>Pairs of prime number</h1>

<p>Print all pairs (sets) of prime numbers (p,q) such that p*q <= n, where n is given number.</p>

<h2>Input:</h2>
<p>The first line of input contains an integer T denoting the number of test cases. T testcases follow.
The first line of each test case is N.</p>

<h2>Output:</h2>
<p>For each testcase, in a new line, print the all pairs of prime numbers in increasing order.</p>

<h2>Constraints:</h2>
<p>1 ≤ T ≤ 50<br>
4 ≤ N ≤ 400</p>

<h2>Example:</h2>
<p>Input:<br>
2<br>
4<br>
8<br>
Output:<br>
2 2<br>
2 2 2 3 3 2</p>

<h2>Explanation:</h2>
<p>Testcase 1: Pair (2, 2) which has both prime numbers as well as satisfying the condition 2*2 <= 4.</p>
